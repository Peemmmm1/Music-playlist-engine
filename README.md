# 🎵 Music Playlist Engine

A music playlist management system built with C, implementing core Data Structures and Algorithms.

## 👥 Team Members
- Pukan — Linked List + Hash Table
- Kwa — Stack + Weighted Shuffle
- Peem — Graph + BFS + Menu

## 📦 Data Structures Used
- **Doubly Linked List** — stores and navigates the playlist
- **Stack** — stores play history for previous track
- **Hash Table** — fast song search by title
- **Graph** — connects similar songs by genre/artist

## ⚙️ Algorithms Used
- **Weighted Shuffle** — randomizes songs based on play count
- **BFS (Breadth-First Search)** — recommends similar songs
